# GRAPHIXPERT Website

Modern, neon-themed website for GRAPHIXPERT offering Services, Projects, and Courses.

## Features

- **Dynamic Content**: Services, Projects, and Courses can be managed via the Admin Dashboard.
- **Admin Dashboard**: Secure area to Add/Edit/Delete content.
- **Neon Aesthetics**: custom CSS and Framer Motion for a premium, glowing orange theme.
- **Responsive Design**: Fully works on mobile and desktop.

## Admin Access

- **Login URL**: `/admin/login` (or click the shield icon in the navbar)
- **Username**: `admin`
- **Password**: `admin123`

## Tech Stack

- React + Vite
- Tailwind-like custom CSS (Vanilla CSS variables)
- Framer Motion (Animations)
- Lucide React (Icons)
- React Router DOM (Routing)

## Development

Run the development server:
```bash
npm run dev
```
